# PokéDex + Team Builder

Pokédex completo y Team Builder interactivo usando la PokeAPI. Explora más de 1000 Pokémon de todas las generaciones, filtra por tipo y arma tu equipo ideal con análisis de cobertura de tipos.

## Stack

- React 19 + Vite + TypeScript
- TailwindCSS — estética gamer con colores dinámicos por tipo
- TanStack Query — caché automático de PokeAPI
- Zustand — equipo persistido en localStorage
- React Router DOM — navegación entre páginas
- Framer Motion — animaciones

## Funcionalidades

- **Pokédex** — lista paginada de 1025 Pokémon, búsqueda por nombre y filtro por tipo
- **Detalle** — stats, tipos, evoluciones, movimientos y habilidades por Pokémon
- **Team Builder** — arma tu equipo de 6, análisis de debilidades y coberturas de tipo en tiempo real
- Equipo persiste en localStorage entre sesiones

## Uso

```
npm install
npm run dev   # http://localhost:4001
npm run test  # tests
npm run build # producción
```
